# Nature Fit iOS Offline Menu App

Premium offline menu and AI meal plan assistant for Nature Fit Restaurant.

## Files

- `index.html` — main iPhone/iPad web app
- `manifest.webmanifest` — PWA manifest
- `sw.js` — offline service worker
- `icons/naturefit.svg` — app icon

## iOS use

1. Host with GitHub Pages.
2. Open the live URL in Safari on iPhone or iPad.
3. Tap Share.
4. Tap Add to Home Screen.
5. Open once online; after that it can run offline.

## Data storage

This offline build saves AI plan leads and feedback into the device browser storage. Use the Export CSV buttons in the app to download saved leads and feedback.

## Notes

This nutrition assistant is a guide, not medical advice. For diabetes, kidney disease, pregnancy, eating disorders, or clinical conditions, consult a qualified clinician or dietitian.
